```javascript
import React from 'react';

// Reusable Button Component
const Button = ({ onClick, label, style, disabled }) => {
  return (
    <button
      onClick={onClick}
      style={{
        padding: '10px 20px',
        backgroundColor: '#007bff',
        color: '#fff',
        border: 'none',
        borderRadius: '5px',
        cursor: 'pointer',
        ...(style || {}),
      }}
      disabled={disabled}
    >
      {label}
    </button>
  );
};

// Example usage of Button component
const App = () => {
  const handleClick = () => {
    alert('Button clicked!');
  };

  return (
    <div>
      <h1>My React App</h1>
      <Button
        onClick={handleClick}
        label="Click Me"
        style={{ backgroundColor: '#28a745' }}
      />
    </div>
  );
};

export default App;
```

### Explanation:
- This code defines a reusable `Button` component that accepts `onClick`, `label`, `style`, and `disabled` as props.
- The button is styled with default styles but allows custom styles to be passed via the `style` prop.
- The `App` component demonstrates how to use the `Button`, showing an alert when the button is clicked.
