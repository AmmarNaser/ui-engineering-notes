This snippet demonstrates how to create a reusable modal component in React. The modal can be opened and closed, and it can accept any content as its children.

### `Modal.js` - Reusable Modal Component

```jsx
import React from 'react';
import './Modal.css'; // Ensure to create a Modal.css for styling

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={e => e.stopPropagation()}>
        <button className="close-button" onClick={onClose}>X</button>
        {children}
      </div>
    </div>
  );
};

export default Modal;
```

### Usage Example

```jsx
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isOpen, setIsOpen] = useState(false);

  const openModal = () => setIsOpen(true);
  const closeModal = () => setIsOpen(false);

  return (
    <div>
      <button onClick={openModal}>Open Modal</button>
      <Modal isOpen={isOpen} onClose={closeModal}>
        <h2>This is a Modal</h2>
        <p>You can put any content here!</p>
      </Modal>
    </div>
  );
};

export default App;
```

- **Reusable Component**: This modal can accept any children, making it flexible for various use cases.
