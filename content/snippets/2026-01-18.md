This snippet demonstrates how to create a reusable modal component in React. It includes basic features such as opening and closing the modal, along with customizable content.

### `Modal.js` - Reusable Modal Component

```javascript
import React, { useState } from 'react';
import './Modal.css'; // Include your styles here

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay">
      <div className="modal-content">
        <button className="close-button" onClick={onClose}>
          &times;
        </button>
        {children}
      </div>
    </div>
  );
};

export default Modal;
```

### Usage Example

```javascript
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setIsModalOpen] = useState(false);

  const toggleModal = () => {
    setIsModalOpen(!isModalOpen);
  };

  return (
    <div>
      <button onClick={toggleModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={toggleModal}>
        <h2>Modal Title</h2>
        <p>This is the modal content.</p>
      </Modal>
    </div>
  );
};

export default App;
```

### Explanation
- **Reusable Component**: The `Modal` component can be reused across different parts of your application.
- **Customizable Content**: It accepts `children` prop, allowing you to pass any content into the modal.
- **Control Visibility**: The `isOpen` prop controls whether the modal is visible, while the `onClose` prop allows you to define the behavior when the modal is closed.

This simple yet effective modal component can enhance the user experience by providing a clear way to display important information or prompts.
