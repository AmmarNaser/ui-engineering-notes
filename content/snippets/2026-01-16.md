This snippet creates a reusable modal component in React that can be easily utilized throughout your application. It includes features for opening and closing the modal, as well as passing in custom content.

### `Modal.js` - Reusable Modal Component

```jsx
import React, { useState } from 'react';
import PropTypes from 'prop-types';
import './Modal.css'; // Make sure to create a CSS file for styling

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay">
      <div className="modal-content">
        <button className="close-modal" onClick={onClose}>
          &times;
        </button>
        {children}
      </div>
    </div>
  );
};

Modal.propTypes = {
  isOpen: PropTypes.bool.isRequired,
  onClose: PropTypes.func.isRequired,
  children: PropTypes.node.isRequired,
};

export default Modal;
```

### Usage Example

```jsx
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setModalOpen] = useState(false);

  const openModal = () => setModalOpen(true);
  const closeModal = () => setModalOpen(false);

  return (
    <div>
      <button onClick={openModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={closeModal}>
        <h1>Hello World</h1>
        <p>This is a reusable modal component!</p>
      </Modal>
    </div>
  );
};

export default App;
```

- **Custom Content**: You can pass any children components to be displayed inside the modal, allowing for flexible usage.
