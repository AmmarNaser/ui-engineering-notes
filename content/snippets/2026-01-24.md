This snippet creates a reusable custom hook for managing form state in React. It simplifies form handling by providing methods to manage field values and handle submission.

### `useForm.js` - Custom Form Handling Hook

```javascript
import { useState } from 'react';

const useForm = (initialValues) => {
  const [values, setValues] = useState(initialValues);
  
  const handleChange = (e) => {
    const { name, value } = e.target;
    setValues({
      ...values,
      [name]: value,
    });
  };

  const handleSubmit = (callback) => (e) => {
    e.preventDefault();
    callback(values);
  };

  return {
    values,
    handleChange,
    handleSubmit,
  };
};

export default useForm;
```

### Usage Example

```javascript
import React from 'react';
import useForm from './useForm';

const MyForm = () => {
  const { values, handleChange, handleSubmit } = useForm({ name: '', email: '' });

  const submitForm = (formData) => {
    console.log('Form Submitted:', formData);
  };

  return (
    <form onSubmit={handleSubmit(submitForm)}>
      <label>
        Name:
        <input
          type="text"
          name="name"
          value={values.name}
          onChange={handleChange}
        />
      </label>
      <label>
        Email:
        <input
          type="email"
          name="email"
          value={values.email}
          onChange={handleChange}
        />
      </label>
      <button type="submit">Submit</button>
    </form>
  );
};

export default MyForm;
```

### Explanation
- **Custom Hook**: `useForm` is a reusable custom hook that manages form state and provides necessary functionalities for field management.
- **State Management**: The hook uses `useState` to keep track of input field values, allowing easy updates on user input.
- **Form Submission**: The `handleSubmit` function wraps the form submission logic, ensuring that form values are passed to the provided callback function when the form is submitted.

This hook can be utilized in any form component, maintaining clean and manageable code while enhancing form handling.
