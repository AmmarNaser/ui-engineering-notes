This snippet demonstrates how to create a reusable button component in React that supports different styles and click handlers through props.

### `Button.js` - Reusable Button Component

```jsx
import React from 'react';
import PropTypes from 'prop-types';

const Button = ({ label, onClick, styleType }) => {
  const baseStyle = 'px-4 py-2 rounded focus:outline-none';
  const styleMap = {
    primary: 'bg-blue-500 text-white hover:bg-blue-700',
    secondary: 'bg-gray-500 text-white hover:bg-gray-700',
    danger: 'bg-red-500 text-white hover:bg-red-700',
  };
  
  return (
    <button 
      className={`${baseStyle} ${styleMap[styleType] || styleMap.primary}`} 
      onClick={onClick}
    >
      {label}
    </button>
  );
};

Button.propTypes = {
  label: PropTypes.string.isRequired,
  onClick: PropTypes.func.isRequired,
  styleType: PropTypes.oneOf(['primary', 'secondary', 'danger']),
};

export default Button;
```

### Usage Example

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import Button from './Button';

const App = () => {
  const handleClick = () => {
    alert('Button clicked!');
  };

  return (
    <div>
      <h1>My App</h1>
      <Button label="Primary Button" onClick={handleClick} styleType="primary" />
      <Button label="Secondary Button" onClick={handleClick} styleType="secondary" />
      <Button label="Danger Button" onClick={handleClick} styleType="danger" />
    </div>
  );
};

ReactDOM.render(<App />, document.getElementById('root'));
```

### Explanation
- **Reusable Component**: The `Button` component is designed to be reusable with customizable props for label, click handler, and style.
- **Prop Types**: Usage of `PropTypes` ensures that the correct types of props are passed, contributing to better code maintainability.
- **Dynamic Styles**: The component can render different styles based on the `styleType` prop, enabling versatility in design while maintaining consistent functionality.
