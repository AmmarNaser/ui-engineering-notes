This snippet demonstrates how to create a reusable modal component in React. The modal can be toggled with a button click and supports custom content.

### `Modal.js` - Reusable Modal Component

```javascript
import React, { useState } from 'react';
import './Modal.css'; // Make sure to create this CSS file for styling

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay">
      <div className="modal">
        <button className="close-button" onClick={onClose}>
          Close
        </button>
        {children}
      </div>
    </div>
  );
};

const App = () => {
  const [isModalOpen, setModalOpen] = useState(false);

  const toggleModal = () => {
    setModalOpen(!isModalOpen);
  };

  return (
    <div>
      <button onClick={toggleModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={toggleModal}>
        <h2>This is a Modal</h2>
        <p>You can add any content here.</p>
      </Modal>
    </div>
  );
};

export default App;
```

### Usage Example

```javascript
import React from 'react';
```

### Explanation
- **isOpen Prop**: Controls the visibility of the modal. If `true`, the modal is shown.
- **onClose Prop**: A function that is called to close the modal, typically passed from the parent component.
- **Children Prop**: Allows passing any content or components to be rendered inside the modal.

This reusable modal component simplifies the process of adding modals to your React applications, providing key functionality with minimal setup.
