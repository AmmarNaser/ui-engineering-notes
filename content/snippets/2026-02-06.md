This snippet demonstrates how to create a reusable customizable button component in React that can serve various purposes in an application, such as submitting forms or triggering actions.

### `CustomButton.js` - Reusable Custom Button Component

```javascript
import React from 'react';

const CustomButton = ({ label, onClick, type = 'button', style, disabled = false }) => {
  return (
    <button 
      type={type} 
      onClick={onClick} 
      style={{ padding: '10px 20px', borderRadius: '5px', ...style }} 
      disabled={disabled}
    >
      {label}
    </button>
  );
};

export default CustomButton;
```

### Usage Example

```javascript
import React from 'react';
import ReactDOM from 'react-dom';
import CustomButton from './CustomButton';

const App = () => {
  const handleClick = () => {
    alert('Button clicked!');
  };

  return (
    <div>
      <h1>Custom Button Example</h1>
      <CustomButton 
        label="Click Me" 
        onClick={handleClick} 
        style={{ backgroundColor: 'blue', color: 'white' }} 
      />
      <CustomButton 
        label="Disabled Button" 
        onClick={handleClick} 
        disabled={true} 
      />
    </div>
  );
};

ReactDOM.render(<App />, document.getElementById('root'));
```

### Explanation
- **Customizable Props**: The `CustomButton` component accepts several props such as `label`, `onClick`, `type`, `style`, and `disabled`, allowing for easy customization.
- **Styling**: The `style` prop allows you to pass a custom style object, making it easy to modify the appearance of the button without altering the component code.
- **Disabled State**: The button can be disabled by passing the `disabled` prop, preventing the user from interacting with it, which is useful for form submissions or preventing duplicate actions.

This component can greatly enhance code reusability and maintainability in a React application.
