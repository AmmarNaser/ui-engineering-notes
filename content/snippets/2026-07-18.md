This snippet provides a reusable and clean button component for a React application, allowing for customizable styles, text, and click handlers. 

### `CustomButton.js` - Reusable Custom Button Component

```jsx
import React from 'react';
import PropTypes from 'prop-types';

const CustomButton = ({ onClick, label, style }) => {
  return (
    <button onClick={onClick} style={style} className="custom-button">
      {label}
    </button>
  );
};

CustomButton.propTypes = {
  onClick: PropTypes.func.isRequired,
  label: PropTypes.string.isRequired,
  style: PropTypes.object,
};

CustomButton.defaultProps = {
  style: {},
};

export default CustomButton;
```

### Usage Example

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import CustomButton from './CustomButton';

const App = () => {
  const handleClick = () => {
    alert('Button clicked!');
  };

  return (
    <div>
      <h1>Hello, World!</h1>
      <CustomButton 
        onClick={handleClick} 
        label="Click Me" 
        style={{ backgroundColor: 'blue', color: 'white', padding: '10px 20px', border: 'none', borderRadius: '5px' }} 
      />
    </div>
  );
};

ReactDOM.render(<App />, document.getElementById('root'));
```

### Explanation
- **Button Customization**: The `CustomButton` component accepts a `label`, allowing you to set the button text easily.
- **Click Handling**: The `onClick` prop enables custom click functionality by passing a function as a prop.
- **Inline Styles**: You can pass an optional `style` prop to apply custom styles directly to the button, enhancing flexibility.

This component can be easily reused and styled across your application.
