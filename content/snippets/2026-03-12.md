This snippet demonstrates how to create a reusable modal component in React that can be easily implemented in your application.

### `Modal.js` - Reusable Modal Component

```javascript
import React from 'react';
import './Modal.css'; // Optional: for styling

const Modal = ({ isOpen, onClose, title, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={(e) => e.stopPropagation()}>
        <h2>{title}</h2>
        {children}
        <button onClick={onClose}>Close</button>
      </div>
    </div>
  );
};

export default Modal;
```

### Usage Example

```javascript
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setModalOpen] = useState(false);

  const toggleModal = () => {
    setModalOpen(!isModalOpen);
  };

  return (
    <div>
      <h1>My App</h1>
      <button onClick={toggleModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={toggleModal} title="My Modal Title">
        <p>This is the content of the modal!</p>
      </Modal>
    </div>
  );
};

export default App;
```

### Explanation
- **Reusable Component**: The `Modal` component can be used anywhere in your application, making it versatile for various purposes.
- **Controlled Visibility**: The modal's visibility is controlled via the `isOpen` prop, allowing seamless integration with parent state.
- **Event Propagation**: Clicking on the modal overlay will close the modal, but clicking inside the modal content won't trigger the close, improving user experience.
- **Customizable Content**: You can pass any content to the modal as children, making it flexible for different use cases.
