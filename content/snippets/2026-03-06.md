This snippet creates a reusable modal component in React that can be easily integrated into any application. The modal supports basic functionalities such as opening, closing, and can display custom content.

### `Modal.js` - Reusable Modal Component

```javascript
import React from 'react';
import './Modal.css'; // Assume there is some basic CSS for styling

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={(e) => e.stopPropagation()}>
        <button className="close-button" onClick={onClose}>
          &times;
        </button>
        {children}
      </div>
    </div>
  );
};

export default Modal;
```

### Usage Example

```javascript
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setModalOpen] = useState(false);

  const toggleModal = () => setModalOpen(!isModalOpen);

  return (
    <div>
      <h1>Welcome to My App</h1>
      <button onClick={toggleModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={toggleModal}>
        <h2>Modal Title</h2>
        <p>This is a reusable modal component!</p>
        <button onClick={toggleModal}>Close Modal</button>
      </Modal>
    </div>
  );
};

export default App;
```

### Explanation
- **Reusable Modal**: The `Modal` component can be used in different parts of the application by passing different children, making it highly reusable.
- **Event Handling**: It captures click events to close the modal using the overlay and a close button, while preventing event bubbling from the modal content.
- **Conditional Rendering**: The modal is rendered conditionally based on the `isOpen` prop, ensuring it only appears when needed.

This modal component can greatly enhance user experience in your application by providing a clean way to present additional information or options.
