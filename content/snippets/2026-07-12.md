This snippet demonstrates how to create a reusable button component in React that can handle different styles and click events.

### `ReusableButton.js` - A Reusable Button Component

```jsx
import React from 'react';
import PropTypes from 'prop-types';

const ReusableButton = ({ label, onClick, styleType }) => {
  const styles = {
    primary: {
      backgroundColor: '#007bff',
      color: '#fff',
      padding: '10px 20px',
      border: 'none',
      borderRadius: '5px',
      cursor: 'pointer',
    },
    secondary: {
      backgroundColor: '#6c757d',
      color: '#fff',
      padding: '10px 20px',
      border: 'none',
      borderRadius: '5px',
      cursor: 'pointer',
    },
    // Add more styles as needed
  };

  return (
    <button style={styles[styleType]} onClick={onClick}>
      {label}
    </button>
  );
};

ReusableButton.propTypes = {
  label: PropTypes.string.isRequired,
  onClick: PropTypes.func.isRequired,
  styleType: PropTypes.oneOf(['primary', 'secondary']).isRequired,
};

export default ReusableButton;
```

### Usage Example

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import ReusableButton from './ReusableButton';

const App = () => {
  const handleClick = () => {
    alert('Button clicked!');
  };

  return (
    <div>
      <h1>Reusable Button Example</h1>
      <ReusableButton 
        label="Primary Button" 
        onClick={handleClick} 
        styleType="primary" 
      />
      <ReusableButton 
        label="Secondary Button" 
        onClick={handleClick} 
        styleType="secondary" 
      />
    </div>
  );
};

ReactDOM.render(<App />, document.getElementById('root'));
```

- **Reusable Component**: The `ReusableButton` can be reused throughout your application with different labels and styles.
- **Style Flexibility**: You can easily expand the `styles` object to include additional styles for different button types.
- **Prop Types**: PropTypes are used to enforce type checking on the component's props for better maintainability.
- **Event Handling**: The button supports custom click events, allowing for versatile functionality.

With this component, you have a solid foundation for buttons in your React application.
