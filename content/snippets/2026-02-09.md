This code snippet demonstrates how to create a simple reusable modal component in React using hooks. The modal can be easily opened and closed with customizable content.

### `Modal.js` - Reusable Modal Component

```jsx
import React, { useState } from 'react';
import './Modal.css'; // Import your modal styles

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={e => e.stopPropagation()}>
        <button className="modal-close" onClick={onClose}>X</button>
        {children}
      </div>
    </div>
  );
};

const useModal = () => {
  const [isOpen, setIsOpen] = useState(false);
  
  const openModal = () => setIsOpen(true);
  const closeModal = () => setIsOpen(false);

  return { isOpen, openModal, closeModal };
};

export { Modal, useModal };
```

### Usage Example

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import { Modal, useModal } from './Modal';

const App = () => {
  const { isOpen, openModal, closeModal } = useModal();

  return (
    <div>
      <h1>My App</h1>
      <button onClick={openModal}>Open Modal</button>
      
      <Modal isOpen={isOpen} onClose={closeModal}>
        <h2>Modal Title</h2>
        <p>This is some content inside the modal.</p>
        <button onClick={closeModal}>Close</button>
      </Modal>
    </div>
  );
};

ReactDOM.render(<App />, document.getElementById('root'));
```

### Explanation
- **Modal Component**: A reusable component that displays content in an overlay. It requires `isOpen` to determine visibility and `onClose` to handle closing.
- **useModal Hook**: A custom hook that manages modal open and close states. It provides functions to open and close the modal.
- **Styling**: Use CSS to style the modal overlay and content for better user experience.

This snippet provides a solid foundation for implementing modals in your React application, making it modular and easy to use.
