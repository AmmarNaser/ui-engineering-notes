This code snippet demonstrates a reusable React component that implements a simple toggle switch. It's useful for enabling/disabling features in a user interface.

### `ToggleSwitch.js` - Reusable Toggle Switch Component

```jsx
import React, { useState } from 'react';

const ToggleSwitch = ({ initialState = false, onToggle }) => {
  const [isOn, setIsOn] = useState(initialState);

  const handleToggle = () => {
    const newState = !isOn;
    setIsOn(newState);
    onToggle(newState);
  };

  return (
    <div onClick={handleToggle} style={{ cursor: 'pointer', width: '50px', height: '25px', backgroundColor: isOn ? 'green' : 'gray', borderRadius: '25px', display: 'flex', alignItems: 'center', justifyContent: isOn ? 'flex-end' : 'flex-start', padding: '5px', transition: 'background-color 0.3s' }}>
      <div style={{ width: '20px', height: '20px', backgroundColor: 'white', borderRadius: '50%' }} />
    </div>
  );
};

export default ToggleSwitch;
```

### Usage Example

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import ToggleSwitch from './ToggleSwitch';

const App = () => {
  const handleToggle = (newState) => {
    console.log('Toggle switch is now:', newState ? 'On' : 'Off');
  };

  return (
    <div>
      <h1>Toggle Switch Example</h1>
      <ToggleSwitch initialState={false} onToggle={handleToggle} />
    </div>
  );
};

ReactDOM.render(<App />, document.getElementById('root'));
```

### Explanation
- **Reusable Component**: The `ToggleSwitch` component can be used multiple times throughout an application with different initial states and handlers.
- **State Management**: It manages its own state (`isOn`) and updates the state whenever the switch is toggled.
- **Customizable Behavior**: The `onToggle` prop allows the parent component to execute custom logic every time the toggle switch is toggled, making it easily extensible.
