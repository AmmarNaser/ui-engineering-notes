This code snippet demonstrates how to create a reusable toggle component in React. The toggle component allows users to switch between two states, and it can be customized with different labels.

### `ToggleSwitch.js` - Reusable Toggle Component

```javascript
import React, { useState } from 'react';

const ToggleSwitch = ({ labelOn = "On", labelOff = "Off", initialValue = false }) => {
    const [isOn, setIsOn] = useState(initialValue);

    const handleToggle = () => {
        setIsOn(prevState => !prevState);
    };

    return (
        <div onClick={handleToggle} style={{ cursor: 'pointer', display: 'flex', alignItems: 'center' }}>
            <div style={{
                width: '50px', height: '25px', borderRadius: '15px',
                backgroundColor: isOn ? 'green' : 'gray', position: 'relative', padding: '3px'
            }}>
                <div style={{
                    width: '20px', height: '20px', borderRadius: '50%',
                    backgroundColor: 'white', position: 'absolute',
                    top: '50%', transform: isOn ? 'translate(25%, -50%)' : 'translate(0, -50%)',
                    transition: 'transform 0.2s'
                }} />
            </div>
            <span style={{ marginLeft: '10px' }}>{isOn ? labelOn : labelOff}</span>
        </div>
    );
};

export default ToggleSwitch;
```

### Usage Example

### Explanation
- **Reusable Component**: The `ToggleSwitch` component can be reused throughout your application, allowing you to create multiple toggle switches with different labels and states.
- **Customizable Labels**: You can customize the labels for the on and off states using the `labelOn` and `labelOff` props.
- **Initial Value**: The `initialValue` prop allows you to set the default state of the toggle switch, making it more flexible for various use cases.
