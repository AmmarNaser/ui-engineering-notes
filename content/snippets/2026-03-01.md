This snippet demonstrates a reusable React component for a basic editable text input that allows users to toggle between viewing and editing the content.

### `EditableTextInput.js` - Editable Text Input Component

```jsx
import React, { useState } from 'react';

const EditableTextInput = ({ initialValue, onSave }) => {
    const [isEditing, setIsEditing] = useState(false);
    const [text, setText] = useState(initialValue);

    const handleSave = () => {
        onSave(text);
        setIsEditing(false);
    };

    return (
        <div>
            {isEditing ? (
                <>
                    <input 
                        type="text" 
                        value={text} 
                        onChange={(e) => setText(e.target.value)} 
                    />
                    <button onClick={handleSave}>Save</button>
                </>
            ) : (
                <>
                    <span>{text}</span>
                    <button onClick={() => setIsEditing(true)}>Edit</button>
                </>
            )}
        </div>
    );
};

export default EditableTextInput;
```

```jsx

const App = () => {
    const [value, setValue] = useState('Hello, world!');

    const handleSave = (newValue) => {

    };

    return (

            <EditableTextInput initialValue={value} onSave={handleSave} />
        </div>
    );
```

### Explanation
- **Editable State**: The component uses a state variable `isEditing` to toggle between viewing and editing modes.
- **Initial Value**: It accepts an `initialValue` prop to set the initial text, making it reusable with different starting values.
- **Save Functionality**: When the user saves their changes, the `onSave` callback is triggered, allowing the parent component to handle the new value. 

This component is flexible and can be easily integrated into any React application requiring editable text fields.
