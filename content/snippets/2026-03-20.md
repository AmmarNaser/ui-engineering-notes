This snippet demonstrates how to create a custom reusable button component in React that accepts props for customization. It allows you to specify styles and click behaviors, making it versatile for various use cases.

### `CustomButton.js` - Reusable Custom Button Component

```javascript
import React from 'react';

const CustomButton = ({ label, onClick, style, disabled }) => {
  return (
    <button
      onClick={onClick}
      style={{ padding: '10px 20px', borderRadius: '5px', ...style }}
      disabled={disabled}
    >
      {label}
    </button>
  );
};

export default CustomButton;
```

### Usage Example

```javascript
import React from 'react';
import ReactDOM from 'react-dom';
import CustomButton from './CustomButton';

const App = () => {
  const handleClick = () => {
    alert('Button clicked!');
  };

  return (
    <div>
      <h1>Welcome to My App</h1>
      <CustomButton
        label="Click Me!"
        onClick={handleClick}
        style={{ backgroundColor: 'blue', color: 'white' }}
        disabled={false}
      />
    </div>
  );
};

ReactDOM.render(<App />, document.getElementById('root'));
```

### Explanation
- **Customizable Props**: The `CustomButton` component accepts `label`, `onClick`, `style`, and `disabled` props to customize its appearance and behavior.
- **Inline Styling**: The component allows for inline styles to be merged with default styles for added flexibility.
- **Disabled State**: The button can be disabled based on the `disabled` prop, providing better control over user interactions.

This component is reusable and can easily be integrated into different parts of a React application.
