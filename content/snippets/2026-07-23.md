This snippet demonstrates how to create a reusable modal component in React that can be displayed and hidden easily. It accepts props for content and visibility control.

### `Modal.js` - Reusable Modal Component

```jsx
import React from 'react';
import './Modal.css'; // Include styles for the modal

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay">
      <div className="modal-content">
        <button className="close-button" onClick={onClose}>Close</button>
        {children}
      </div>
    </div>
  );
};

export default Modal;
```

### Usage Example

```jsx
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setModalOpen] = useState(false);

  const toggleModal = () => {
    setModalOpen(prev => !prev);
  };

  return (
    <div>
      <button onClick={toggleModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={toggleModal}>
        <h2>Modal Title</h2>
        <p>This is the modal content!</p>
      </Modal>
    </div>
  );
};

export default App;
```

### Explanation
- **Modal Visibility**: The modal component utilizes `isOpen` prop to determine whether it is visible or not.
- **Close Functionality**: The `onClose` prop is a callback function that is triggered to close the modal.
- **Content Injection**: The modal allows you to inject any children elements, making it versatile for different content needs.

This reusable modal component can be easily integrated into any React application for handling popup dialogs.
