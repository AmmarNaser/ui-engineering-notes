This snippet demonstrates a reusable button component in React that accepts different colors and sizes as props, allowing for easy customization in your application.

### `Button.js` - Reusable Button Component

```jsx
import React from 'react';

const Button = ({ color = 'blue', size = 'medium', children, onClick }) => {
  const styles = {
    backgroundColor: color,
    padding: size === 'small' ? '5px 10px' : size === 'large' ? '15px 20px' : '10px 15px',
    color: 'white',
    border: 'none',
    borderRadius: '5px',
    cursor: 'pointer',
  };

  return (
    <button style={styles} onClick={onClick}>
      {children}
    </button>
  );
};

export default Button;
```

### Usage Example

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import Button from './Button';

const App = () => {
  const handleClick = () => {
    alert('Button clicked!');
  };

  return (
    <div>
      <h1>Welcome to My App</h1>
      <Button color="green" size="small" onClick={handleClick}>Small Green Button</Button>
      <Button color="red" size="medium" onClick={handleClick}>Medium Red Button</Button>
      <Button color="blue" size="large" onClick={handleClick}>Large Blue Button</Button>
    </div>
  );
};

ReactDOM.render(<App />, document.getElementById('root'));
```

### Explanation

- **Custom Colors**: The button can accept any color through the `color` prop, making it adaptable to various design requirements.
- **Variable Size**: The `size` prop allows for three size options (small, medium, large), enabling users to tailor the button’s size according to their layout needs.
- **Event Handling**: The `onClick` prop ensures that any function can be executed when the button is clicked, making it interactive.

This reusable button component is a flexible addition to any React project, allowing for consistent styling and user interaction throughout your application.
