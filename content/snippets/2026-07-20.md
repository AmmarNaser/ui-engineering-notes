This snippet creates a reusable modal component in React that can be easily integrated into your applications. The modal can be customized with title, content, and actions.

### `Modal.js` - Reusable Modal Component

```javascript
import React from 'react';
import PropTypes from 'prop-types';
import './Modal.css'; // Ensure to create corresponding CSS for styling

const Modal = ({ isOpen, onClose, title, children, actions }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay">
      <div className="modal-content">
        <h2>{title}</h2>
        <div>{children}</div>
        <div className="modal-actions">
          {actions.map((action, index) => (
            <button key={index} onClick={action.onClick}>
              {action.label}
            </button>
          ))}
        </div>
        <button className="close-button" onClick={onClose}>Close</button>
      </div>
    </div>
  );
};

Modal.propTypes = {
  isOpen: PropTypes.bool.isRequired,
  onClose: PropTypes.func.isRequired,
  title: PropTypes.string.isRequired,
  children: PropTypes.node,
  actions: PropTypes.arrayOf(PropTypes.shape({
    label: PropTypes.string.isRequired,
    onClick: PropTypes.func.isRequired,
  })).isRequired,
};

export default Modal;
```

### Usage Example

```javascript
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setModalOpen] = useState(false);

  const handleOpenModal = () => setModalOpen(true);
  const handleCloseModal = () => setModalOpen(false);

  const actions = [
    {
      label: 'Confirm',
      onClick: () => { alert("Confirmed!"); handleCloseModal(); },
    },
    {
      label: 'Cancel',
      onClick: handleCloseModal,
    },
  ];

  return (
    <div>
      <button onClick={handleOpenModal}>Open Modal</button>
      <Modal
        isOpen={isModalOpen}
        onClose={handleCloseModal}
        title="Sample Modal"
        actions={actions}
      >
        This is a sample modal content!
      </Modal>
    </div>
  );
};

export default App;
```

### Explanation
- **isOpen**: A boolean prop that controls whether the modal is visible or not.
- **onClose**: A function prop that defines the action to be taken when the modal is closed.
- **title**: A string prop to set the title of the modal.
- **children**: Allows for custom content inside the modal.
- **actions**: An array of action objects that specify buttons to be shown with their labels and click handlers.

This modal component is both flexible and easy to use, making it a great addition to your UI toolkit!
