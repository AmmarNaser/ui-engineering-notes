This snippet demonstrates a reusable button component in React that can handle click events and accept custom styles.

### `Button.js` - Reusable Button Component

```jsx
import React from 'react';

// Button Component
const Button = ({ onClick, style, children }) => {
  return (
    <button onClick={onClick} style={style}>
      {children}
    </button>
  );
};

export default Button;
```

### Usage Example

```jsx
import React from 'react';
import Button from './Button';

const App = () => {
  const handleClick = () => {
    alert('Button was clicked!');
  };

  return (
    <div>
      <h1>Welcome to My App</h1>
      <Button onClick={handleClick} style={{ padding: '10px', backgroundColor: 'blue', color: 'white' }}>
        Click Me
      </Button>
    </div>
  );
};

export default App;
```

### Explanation
- **Custom Styling**: The Button component accepts a `style` prop that allows users to pass in custom CSS styles directly.
- **Children Prop**: It supports passing any React node as children, making it flexible for different kinds of button labels or inner components.
- **Click Event Handling**: The component efficiently handles click events through the `onClick` prop, making it easy to link to various functionalities.
