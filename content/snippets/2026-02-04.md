This snippet demonstrates how to create a reusable button component in React. It allows for customizable styles and actions by passing props.

### `Button.js` - Reusable Button Component

```javascript
import React from 'react';

const Button = ({ label, onClick, style, disabled }) => {
  return (
    <button
      onClick={onClick}
      style={{ padding: '10px 20px', ...style }}
      disabled={disabled}
    >
      {label}
    </button>
  );
};

export default Button;
```

### Usage Example

```javascript
import React from 'react';
import ReactDOM from 'react-dom';
import Button from './Button';

const App = () => {
  const handleClick = () => {
    alert('Button clicked!');
  };

  return (
    <div>
      <h1>Button Component Example</h1>
      <Button
        label="Click Me!"
        onClick={handleClick}
        style={{ backgroundColor: 'blue', color: 'white' }}
      />
      <Button
        label="Disabled Button"
        onClick={handleClick}
        disabled={true}
        style={{ backgroundColor: 'gray', color: 'white' }}
      />
    </div>
  );
};

ReactDOM.render(<App />, document.getElementById('root'));
```

### Explanation
- **Customizable Styles**: The style of the button can be easily modified by passing a `style` prop, allowing for versatility in design.
- **Disabled State**: The button can be disabled using the `disabled` prop to prevent user interaction.
- **Click Handler**: You can define custom click behavior using the `onClick` prop, making the component flexible for various use cases.

This reusable button component makes it easy to implement uniform styling and behavior throughout your React application.
