This snippet demonstrates how to create a reusable modal component in React for displaying info or confirmation to users.

### `Modal.js` - Reusable Modal Component

```jsx
import React from 'react';
import PropTypes from 'prop-types';
import './Modal.css'; // Import your CSS for modal styling

const Modal = ({ isOpen, onClose, title, children }) => {
  if (!isOpen) {
    return null;
  }

  return (
    <div className="modal-overlay">
      <div className="modal-content">
        <h2>{title}</h2>
        {children}
        <button onClick={onClose}>Close</button>
      </div>
    </div>
  );
};

Modal.propTypes = {
  isOpen: PropTypes.bool.isRequired,
  onClose: PropTypes.func.isRequired,
  title: PropTypes.string.isRequired,
  children: PropTypes.node,
};

export default Modal;
```

### Usage Example

```jsx
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setModalOpen] = useState(false);

  const openModal = () => setModalOpen(true);
  const closeModal = () => setModalOpen(false);

  return (
    <div>
      <h1>Welcome to My App</h1>
      <button onClick={openModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={closeModal} title="My Modal">
        <p>This is the content of the modal!</p>
      </Modal>
    </div>
  );
};

export default App;
```

### Explanation
- **Reusable Component**: The `Modal` component can be used throughout your application to display different content by passing props.
- **Dynamic Visibility**: The modal only renders when the `isOpen` prop is true, making it efficient and responsive to state changes.
- **Custom Close Behavior**: The `onClose` prop allows you to define the behavior when the modal is closed, enhancing reusability.
