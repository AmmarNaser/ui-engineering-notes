This snippet demonstrates how to create a simple reusable modal component in React using hooks. The modal can be opened and closed programmatically, making it easy to integrate into any application.

### `Modal.js` - Reusable Modal Component

```javascript
import React, { useState } from 'react';
import PropTypes from 'prop-types';
import './Modal.css'; // Import your CSS styles

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={(e) => e.stopPropagation()}>
        <span className="close-btn" onClick={onClose}>&times;</span>
        {children}
      </div>
    </div>
  );
};

Modal.propTypes = {
  isOpen: PropTypes.bool.isRequired,
  onClose: PropTypes.func.isRequired,
  children: PropTypes.node.isRequired,
};

export default Modal;
```

### Usage Example

```javascript
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setIsModalOpen] = useState(false);

  const handleOpenModal = () => {
    setIsModalOpen(true);
  };

  const handleCloseModal = () => {
    setIsModalOpen(false);
  };

  return (
    <div>
      <button onClick={handleOpenModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={handleCloseModal}>
        <h2>Hello, World!</h2>
        <p>This is a reusable modal component.</p>
        <button onClick={handleCloseModal}>Close</button>
      </Modal>
    </div>
  );
};

export default App;
```

### Explanation
- **Reusable Component**: The `Modal` can be used anywhere in your application by importing it and passing in the necessary props.
- **Easy Visibility Control**: The `isOpen` prop helps manage the visibility of the modal effortlessly, making it simple to show or hide based on state.
- **Click Outside to Close**: Clicking outside the modal content will close the modal, enhancing user experience.
- **Customizable Content**: You can pass any children to the modal, allowing for flexible content management.
