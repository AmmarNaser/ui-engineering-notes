This React component provides a responsive modal dialog that can be easily integrated into any application. It allows users to input text and submit the form.

### `ModalDialog.js` - Responsive Modal Dialog Component

```jsx
import React, { useState } from 'react';
import './ModalDialog.css'; // Ensure to create this CSS file for styling

const ModalDialog = ({ isOpen, onClose, title, onSubmit }) => {
  const [inputValue, setInputValue] = useState('');

  const handleSubmit = (e) => {
    e.preventDefault();
    if (inputValue.trim()) {
      onSubmit(inputValue);
      setInputValue('');
      onClose();
    }
  };

  if (!isOpen) return null;

  return (
    <div className="modal-overlay">
      <div className="modal-content">
        <h2>{title}</h2>
        <form onSubmit={handleSubmit}>
          <input
            type="text"
            value={inputValue}
            onChange={(e) => setInputValue(e.target.value)}
            placeholder="Enter text..."
            required
          />
          <button type="submit">Submit</button>
          <button type="button" onClick={onClose}>Close</button>
        </form>
      </div>
    </div>
  );
};

export default ModalDialog;
```

### Usage Example
```jsx
import React, { useState } from 'react';
import ModalDialog from './ModalDialog';

const App = () => {
  const [isModalOpen, setModalOpen] = useState(false);

  const handleModalOpen = () => setModalOpen(true);
  const handleModalClose = () => setModalOpen(false);
  const handleSubmit = (value) => {
    console.log('Submitted value:', value);
  };

  return (
    <div>
      <button onClick={handleModalOpen}>Open Modal</button>
      <ModalDialog
        isOpen={isModalOpen}
        onClose={handleModalClose}
        title="My Modal"
        onSubmit={handleSubmit}
      />
    </div>
  );
};

export default App;
```

### Explanation
- **Component Structure**: The `ModalDialog` component accepts props for open state, title, and a submit callback.
- **State Management**: The component uses the `useState` hook to manage input value and respond to user input.
- **Form Submission**: When the form is submitted, the input value is passed to the callback function and the modal closes.
- **Conditional Rendering**: The modal only renders if `isOpen` is true, preventing unnecessary DOM elements.
- **Styling**: Ensure you create a `ModalDialog.css` file to style the modal overlay and content.
