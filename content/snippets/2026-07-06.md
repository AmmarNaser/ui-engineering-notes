This snippet demonstrates a reusable React component for a modal. The modal can be opened and closed, and it displays any content passed to it as children.

### `Modal.js` - Reusable Modal Component

```javascript
import React from 'react';
import './Modal.css'; // Assume some basic styles are defined in Modal.css

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={(e) => e.stopPropagation()}>
        <button className="modal-close" onClick={onClose}>×</button>
        {children}
      </div>
    </div>
  );
};

export default Modal;
```

### Usage Example

```javascript
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setModalOpen] = useState(false);

  const toggleModal = () => {
    setModalOpen((prev) => !prev);
  };

  return (
    <div>
      <h1>Welcome to the Modal Demo</h1>
      <button onClick={toggleModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={toggleModal}>
        <h2>Modal Title</h2>
        <p>This is the content of the modal!</p>
        <button onClick={toggleModal}>Close Modal</button>
      </Modal>
    </div>
  );
};

export default App;
```

### Explanation
- **Reusable Component**: The `Modal` component can be reused throughout the application by passing different content as children.
- **Click Outside to Close**: The modal closes when the overlay is clicked, providing an intuitive user experience.
- **Stop Propagation**: The modal content area ignores overlay clicks, ensuring the modal stays open when interacting with its content.

This modal component is flexible and can be easily integrated into any React application. Adjust styles in `Modal.css` to suit your design preferences.
