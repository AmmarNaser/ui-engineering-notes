This snippet implements a simple modal component in React that can be reused throughout your application for various purposes.

### `Modal.js` - Reusable Modal Component

```javascript
import React from 'react';
import PropTypes from 'prop-types';
import './Modal.css'; // Make sure to create this CSS file for modal styling

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={(e) => e.stopPropagation()}>
        <button className="close-button" onClick={onClose}>X</button>
        {children}
      </div>
    </div>
  );
};

Modal.propTypes = {
  isOpen: PropTypes.bool.isRequired,
  onClose: PropTypes.func.isRequired,
  children: PropTypes.node.isRequired,
};

export default Modal;
```

### Usage Example
```javascript
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isOpen, setIsOpen] = useState(false);

  const openModal = () => setIsOpen(true);
  const closeModal = () => setIsOpen(false);

  return (
    <div>
      <button onClick={openModal}>Open Modal</button>
      <Modal isOpen={isOpen} onClose={closeModal}>
        <h2>Hello, World!</h2>
        <p>This is a reusable modal component.</p>
      </Modal>
    </div>
  );
};

export default App;
```

### Explanation
- **Modal Component**: A reusable component that renders its children inside a modal overlay when `isOpen` is true.
- **Close Functionality**: Clicking outside the modal content or on the close button will trigger the `onClose` function to close the modal.
- **Prop Types**: Usage of PropTypes ensures that the correct data types are passed to the component, enhancing reliability.

This modular approach allows you to easily manage and display modals across your application.
