This snippet demonstrates a reusable React component that implements a custom button with loading and disabled states, enhancing user interactivity in applications.

### `CustomButton.js` - Custom Button Component with Loading State

```javascript
import React from 'react';

const CustomButton = ({ onClick, isLoading, disabled, children }) => {
  return (
    <button
      onClick={onClick}
      disabled={isLoading || disabled}
      style={{
        backgroundColor: isLoading ? '#ccc' : '#007bff',
        color: '#fff',
        cursor: isLoading || disabled ? 'not-allowed' : 'pointer',
        padding: '10px 20px',
        border: 'none',
        borderRadius: '5px',
      }}
    >
      {isLoading ? 'Loading...' : children}
    </button>
  );
};

export default CustomButton;
```

### Usage Example

```javascript
import React, { useState } from 'react';
import CustomButton from './CustomButton';

const App = () => {
  const [isLoading, setIsLoading] = useState(false);

  const handleClick = () => {
    setIsLoading(true);
    setTimeout(() => {
      setIsLoading(false);
    }, 2000); // Simulate a network request
  };

  return (
    <div>
      <h1>Custom Button Example</h1>
      <CustomButton onClick={handleClick} isLoading={isLoading}>
        Submit
      </CustomButton>
    </div>
  );
};

export default App;
```

### Explanation
- **Reusable Component**: The `CustomButton` can be used in various parts of your application, accepting props for click handling, loading, and disabled states.
- **Loading State**: When the `isLoading` prop is true, the button displays a loading message and is styled to indicate it cannot be clicked.
- **Dynamic Styling**: Button styles change based on props, providing visual feedback for loading and disabled states.

This component simplifies the implementation of interactive buttons in your projects while maintaining clean and reusable code.
