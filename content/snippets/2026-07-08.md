This snippet demonstrates a reusable modal component built with React that can be toggled on and off. It handles visibility and can display any content you need within the modal.

### `Modal.js` - Reusable React Modal Component

```javascript
import React, { useState } from 'react';
import './Modal.css'; // Include CSS for styling

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={e => e.stopPropagation()}>
        <button className="close-button" onClick={onClose}>×</button>
        {children}
      </div>
    </div>
  );
};

const App = () => {
  const [isModalOpen, setIsModalOpen] = useState(false);

  return (
    <div>
      <button onClick={() => setIsModalOpen(true)}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={() => setIsModalOpen(false)}>
        <h1>Modal Title</h1>
        <p>This is a simple modal example.</p>
      </Modal>
    </div>
  );
};

export default App;
```

### Usage Example

### Explanation
- **Modal Visibility**: The modal's visibility is controlled by the `isOpen` prop, allowing it to be toggled easily within the parent component.
- **Close Functionality**: The modal can be closed by clicking the close button or outside the modal content area, enhancing usability.
- **Reusable Structure**: The modal can display any children passed to it, making it flexible for various content types.
