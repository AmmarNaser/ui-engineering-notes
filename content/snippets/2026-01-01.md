Here's a simple and practical React code snippet for a reusable input component that includes validation and displays an error message if validation fails. This can be used for forms in your frontend applications.

```jsx
import React, { useState } from 'react';

const TextInput = ({ label, type = "text", required = false, validate }) => {
  const [value, setValue] = useState('');
  const [error, setError] = useState('');

  const handleChange = (e) => {
    const newValue = e.target.value;
    setValue(newValue);
    
    if (validate) {
      const validationError = validate(newValue);
      setError(validationError);
    }
  };

  return (
    <div className="text-input">
      <label>
        {label}{required && ' *'}
        <input 
          type={type} 
          value={value} 
          onChange={handleChange} 
          required={required} 
        />
      </label>
      {error && <span className="error-message">{error}</span>}
    </div>
  );
};

// Example of using TextInput with validation
const App = () => {
  const validateEmail = (email) => {
    const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    if (!regex.test(email)) return "Invalid email format";
    return "";
  };

  return (
    <form>
      <TextInput label="Email" type="email" required validate={validateEmail} />
      <button type="submit">Submit</button>
    </form>
  );
};

export default App;
```

1. Define your validation function as needed (e.g., `validateEmail`).
2. Use the `TextInput` component where you need an input field, passing the validation function to it.
