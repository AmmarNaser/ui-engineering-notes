This code snippet demonstrates a reusable React component that manages a simple countdown timer. You can use it in various applications where a countdown feature is needed.

### `CountdownTimer.js` - Simple Countdown Timer Component

```javascript
import React, { useState, useEffect } from 'react';

const CountdownTimer = ({ initialSeconds, onFinish }) => {
  const [seconds, setSeconds] = useState(initialSeconds);

  useEffect(() => {
    if (seconds > 0) {
      const timer = setInterval(() => {
        setSeconds(prev => prev - 1);
      }, 1000);

      return () => clearInterval(timer);
    } else if (onFinish) {
      onFinish();
    }
  }, [seconds, onFinish]);

  return (
    <div>
      <h1>{seconds} seconds remaining</h1>
    </div>
  );
};

export default CountdownTimer;
```

### Explanation
- **State Management**: The component uses the `useState` hook to manage the remaining seconds.
- **Lifetime Management**: It employs `useEffect` to start the timer and clear it when no longer needed, avoiding memory leaks.
- **Callback Function**: The `onFinish` prop allows users to define a function that executes when the countdown finishes.
