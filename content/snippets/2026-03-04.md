This snippet demonstrates a reusable React component for a modal dialog that can be easily integrated into any application. The modal supports basic functionality like opening, closing, and passing custom content.

### `Modal.js` - Reusable Modal Component

```jsx
import React from 'react';
import PropTypes from 'prop-types';
import './Modal.css'; // Include your styles for the modal

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay">
      <div className="modal-content">
        <button className="close-button" onClick={onClose}>
          &times;
        </button>
        {children}
      </div>
    </div>
  );
};

Modal.propTypes = {
  isOpen: PropTypes.bool.isRequired,
  onClose: PropTypes.func.isRequired,
  children: PropTypes.node.isRequired,
};

export default Modal;
```

### Usage Example

```jsx
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setModalOpen] = useState(false);

  const toggleModal = () => {
    setModalOpen(!isModalOpen);
  };

  return (
    <div>
      <h1>Welcome to the Modal Demo</h1>
      <button onClick={toggleModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={toggleModal}>
        <h2>Hello, this is a modal!</h2>
        <p>You can add any content here.</p>
      </Modal>
    </div>
  );
};

export default App;
```

- **Modal Component**: A functional component that takes `isOpen`, `onClose`, and `children` as props to control its visibility and content.
- **Close Button**: A button is included to allow users to close the modal easily.
- **Conditional Rendering**: The modal only renders when `isOpen` is true, enhancing performance and usability.
- **Prop Types**: PropTypes are used to enforce type checking for the props, ensuring the component is used correctly.
