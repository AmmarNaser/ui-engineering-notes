Here is a reusable React component that demonstrates a simple modal dialog. It can be integrated into various applications to display messages or gather user input.

### `Modal.js` - Reusable Modal Component

```javascript
import React, { useEffect } from 'react';
import PropTypes from 'prop-types';
import './Modal.css'; // Ensure proper styling is applied

const Modal = ({ isOpen, onClose, title, children }) => {
  useEffect(() => {
    if (isOpen) {
      document.body.style.overflow = 'hidden';
    } else {
      document.body.style.overflow = 'unset';
    }
  }, [isOpen]);

  if (!isOpen) return null;

  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={(e) => e.stopPropagation()}>
        <h2>{title}</h2>
        {children}
        <button onClick={onClose}>Close</button>
      </div>
    </div>
  );
};

Modal.propTypes = {
  isOpen: PropTypes.bool.isRequired,
  onClose: PropTypes.func.isRequired,
  title: PropTypes.string.isRequired,
  children: PropTypes.node.isRequired,
};

export default Modal;
```

### Usage Example

```javascript
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setIsModalOpen] = useState(false);

  return (
    <div>
      <h1>Welcome to My App</h1>
      <button onClick={() => setIsModalOpen(true)}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={() => setIsModalOpen(false)} title="My Modal">
        <p>This is the content of the modal.</p>
      </Modal>
    </div>
  );
};

export default App;
```

### Explanation
- **Modal Functionality**: This component displays a modal dialog that can be opened and closed by the user.
- **Props Handling**: The component accepts `isOpen`, `onClose`, `title`, and `children` as props, allowing for flexible usage.
- **Close on Overlay Click**: Clicking the overlay will close the modal, while clicks inside the modal content will not close it, enhancing user experience.
- **Body Overflow Control**: The useEffect hook manages the overflow style of the body element when the modal is open, preventing background scrolling.

This modal component is flexible and easily adaptable for various use cases within your React applications.
