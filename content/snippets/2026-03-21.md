This snippet creates a reusable Modal component in React, allowing you to display content in a dialog overlay. It's easy to integrate and manage in any application.

### `Modal.js` - Reusable Modal Component

```jsx
import React, { useEffect } from 'react';
import './Modal.css'; // Ensure to create a Modal.css file for styling

const Modal = ({ isOpen, onClose, children }) => {
  useEffect(() => {
    if (isOpen) {
      document.body.style.overflow = 'hidden';
    } else {
      document.body.style.overflow = 'unset';
    }
    return () => {
      document.body.style.overflow = 'unset';
    };
  }, [isOpen]);

  if (!isOpen) return null;

  return (
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={(e) => e.stopPropagation()}>
        <button className="modal-close" onClick={onClose}>X</button>
        {children}
      </div>
    </div>
  );
};

export default Modal;
```

### Usage Example

```jsx
import React, { useState } from 'react';
import Modal from './Modal';

const App = () => {
  const [isModalOpen, setIsModalOpen] = useState(false);

  const openModal = () => setIsModalOpen(true);
  const closeModal = () => setIsModalOpen(false);

  return (
    <div>
      <button onClick={openModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={closeModal}>
        <h1>This is a Modal</h1>
        <p>Some content goes here.</p>
      </Modal>
    </div>
  );
};

export default App;
```

### Explanation
- **Reusable Component**: The `Modal` component can be reused anywhere in your application, encapsulating its own logic for opening and closing.
- **Event Handling**: It handles clicks to close the modal, either when clicking on the overlay or the close button.
- **Body Overflow Management**: The modal manages the body’s overflow style to prevent scrolling when it’s open, improving user experience.
- **Customizable Content**: You can pass any valid React elements as children to customize the modal content.

This Modal component provides a solid foundation for adding dialog functionality in your React applications.
