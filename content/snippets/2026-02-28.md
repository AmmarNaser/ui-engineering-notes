This snippet demonstrates how to create a reusable button component in React that supports loading states and callbacks for click events.

### `Button.js` - Reusable Button Component with Loading State

```javascript
import React from 'react';

const Button = ({ onClick, isLoading, children, disabled }) => {
  return (
    <button 
      onClick={onClick} 
      disabled={isLoading || disabled} 
      className={`btn ${isLoading ? 'loading' : ''}`}
    >
      {isLoading ? 'Loading...' : children}
    </button>
  );
};

export default Button;
```

### Usage Example

```javascript
import React, { useState } from 'react';
import Button from './Button';

const App = () => {
  const [loading, setLoading] = useState(false);

  const handleClick = () => {
    setLoading(true);
    setTimeout(() => {
      setLoading(false);
      alert('Button Clicked!');
    }, 2000); // Simulating a network request
  };

  return (
    <div>
      <h1>Button Component Example</h1>
      <Button onClick={handleClick} isLoading={loading}>
        Click Me
      </Button>
    </div>
  );
};

export default App;
```

### Explanation
- **Reusable Component**: The `Button` component can be reused throughout your application with different content and click behavior.
- **Loading State**: The button can show a loading state while operations are being performed, providing feedback to the user.
- **Customizable Props**: It accepts props like `onClick`, `isLoading`, `children`, and `disabled` to customize its behavior and appearance.

This button component is ideal for enhancing user experience in your forms or other interactive elements.
