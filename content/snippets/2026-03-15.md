This snippet demonstrates a reusable React component that implements a toggle switch. It's styled for consistency and can be easily integrated into any React application.

### `ToggleSwitch.js` - Reusable Toggle Switch Component

```jsx
import React, { useState } from 'react';
import './ToggleSwitch.css'; // Assuming some basic CSS is applied in this file

const ToggleSwitch = ({ label, isChecked, onToggle }) => {
  const [checked, setChecked] = useState(isChecked);

  const handleToggle = () => {
    setChecked(!checked);
    if (onToggle) onToggle(!checked);
  };

  return (
    <div className="toggle-switch">
      <label>
        {label}
        <input 
          type="checkbox" 
          checked={checked} 
          onChange={handleToggle} 
        />
        <span className="slider"></span>
      </label>
    </div>
  );
};

export default ToggleSwitch;
```
