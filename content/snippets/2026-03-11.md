This snippet demonstrates how to create a simple customizable button component in React that can handle different styles and click events.

### `CustomButton.js` - Reusable Custom Button Component

```jsx
import React from 'react';

const CustomButton = ({ label, onClick, style, type = 'button' }) => {
    return (
        <button 
            type={type} 
            onClick={onClick} 
            style={{ padding: '10px 20px', border: 'none', borderRadius: '5px', cursor: 'pointer', ...style }}
        >
            {label}
        </button>
    );
};

export default CustomButton;
```

### Usage Example

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import CustomButton from './CustomButton';

const App = () => {
    const handleClick = () => {
        alert('Button clicked!');
    };

    return (
        <div>
            <h1>Custom Button Example</h1>
            <CustomButton 
                label="Click Me!" 
                onClick={handleClick} 
                style={{ backgroundColor: 'blue', color: 'white' }} 
            />
        </div>
    );
};

ReactDOM.render(<App />, document.getElementById('root'));
```

### Explanation
- **Customizable Styles**: The button can accept a `style` prop for further customization, allowing it to adapt to your UI needs.
- **Click Handler**: The `onClick` prop allows you to specify a function that will run when the button is clicked, making it reusable for various actions.
- **Flexible Label**: The `label` prop enables you to customize the text displayed on the button, making it suitable for different contexts.

This component is designed to be reusable and fits seamlessly into various projects. Enjoy building your applications!
