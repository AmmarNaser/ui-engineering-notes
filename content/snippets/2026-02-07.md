This snippet demonstrates how to create a reusable toggle component in React. This component allows users to toggle between two states and can be easily integrated into various parts of your application.

### `Toggle.js` - Reusable Toggle Component

```javascript
import React, { useState } from 'react';

const Toggle = ({ initialState = false, labels = ['Off', 'On'] }) => {
  const [toggled, setToggled] = useState(initialState);

  const handleToggle = () => {
    setToggled((prev) => !prev);
  };

  return (
    <div onClick={handleToggle} style={{ cursor: 'pointer', padding: '10px', border: '1px solid #ccc', display: 'inline-block' }}>
      {toggled ? labels[1] : labels[0]}
    </div>
  );
};

export default Toggle;
```

### Usage Example

```javascript
import React from 'react';
import ReactDOM from 'react-dom';
import Toggle from './Toggle';

const App = () => {
  return (
    <div>

    </div>
  );
};

ReactDOM.render(<App />, document.getElementById('root'));
```

### Explanation
- **Reusable Component**: The `Toggle` component is designed to be reusable with any state management needs without being tied to any specific context.
- **Customizable Labels**: The component accepts customizable labels for the two states, making it flexible for different use cases.
- **Initial State Prop**: You can set the initial state of the toggle when you render the component, allowing for dynamic behavior based on props.

This toggle component can be invaluable in interactive UI scenarios, such as switching settings or modes in your application.
