### `useFetch.js` - Custom Hook for Data Fetching

```javascript
import { useState, useEffect } from 'react';

const useFetch = (url) => {
    const [data, setData] = useState(null);
    const [loading, setLoading] = useState(true);
    const [error, setError] = useState(null);

    useEffect(() => {
        const fetchData = async () => {
            try {
                const response = await fetch(url);
                if (!response.ok) {
                    throw new Error('Network response was not ok');
                }
                const result = await response.json();
                setData(result);
            } catch (err) {
                setError(err);
            } finally {
                setLoading(false);
            }
        };

        fetchData();
    }, [url]);

    return { data, loading, error };
};

export default useFetch;
```

### Usage Example

```javascript
import React from 'react';
import useFetch from './useFetch';

const App = () => {
    const { data, loading, error } = useFetch('https://api.example.com/data');

    if (loading) return <div>Loading...</div>;
    if (error) return <div>Error: {error.message}</div>;

    return (
        <div>
            <h1>Data:</h1>
            <pre>{JSON.stringify(data, null, 2)}</pre>
        </div>
    );
};

export default App;
```

### Explanation
- **Custom Hook**: `useFetch` is a reusable hook that you can use anywhere in your application to fetch data from an API endpoint.
- **Loading State**: It handles a loading state, showing "Loading..." while the request is being processed.
- **Error Handling**: It captures errors and displays a user-friendly message if the fetch fails.
