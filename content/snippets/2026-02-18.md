This snippet demonstrates how to create a reusable modal component in React. The modal can be easily opened and closed, and it accepts custom content as props.

### `Modal.js` - Reusable Modal Component

```javascript
import React, { useState } from 'react';
import './Modal.css'; // Assuming you have some basic styles for the modal

const Modal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay">
      <div className="modal-content">
        <button onClick={onClose} className="modal-close-button">X</button>
        {children}
      </div>
    </div>
  );
};

const App = () => {
  const [isModalOpen, setModalOpen] = useState(false);

  const openModal = () => setModalOpen(true);
  const closeModal = () => setModalOpen(false);

  return (
    <div>
      <button onClick={openModal}>Open Modal</button>
      <Modal isOpen={isModalOpen} onClose={closeModal}>
        <h1>Hello, I'm a Modal!</h1>
        <p>This is some modal content.</p>
      </Modal>
    </div>
  );
};

export default App;
```

### Usage Example

```javascript
import React from 'react';

ReactDOM.render(<App />, document.getElementById('root'));
```

### Explanation
- **Reusable Component**: The `Modal` component can be reused throughout the application, accepting custom content and managing its open/close state via props.
- **Conditional Rendering**: The modal is conditionally rendered based on the `isOpen` prop, making it simple to show or hide the modal.
- **Close Functionality**: The modal includes a close button that triggers the `onClose` function, allowing users to dismiss the modal easily.

This modal component is straightforward to implement and adapt for various use cases in your application.
