### Custom Hooks - Creating Reusable Logic in React

Custom hooks are a powerful feature in React that enable developers to extract and reuse stateful logic. They allow you to encapsulate functionality that can be shared across components, improving code organization and reducing duplication.

### When to Use It
- **Component Logic Reusability**: When you have shared logic between multiple components that can benefit from encapsulation to avoid code duplication.
- **Complex State Management**: When managing complex state transitions or effects across various components, especially when that logic is not easily managed within the component itself.

### Example
```javascript
// useFetch.js - custom hook for fetching data
import { useState, useEffect } from 'react';

const useFetch = (url) => {
    const [data, setData] = useState(null);
    const [loading, setLoading] = useState(true);
    const [error, setError] = useState(null);
    
    useEffect(() => {
        const fetchData = async () => {
            try {
                const response = await fetch(url);
                if (!response.ok) throw new Error('Network response was not ok');
                const result = await response.json();
                setData(result);
            } catch (err) {
                setError(err);
            } finally {
                setLoading(false);
            }
        };

        fetchData();
    }, [url]);

    return { data, loading, error };
};

// Sample Component using the custom hook
import React from 'react';
import useFetch from './useFetch';

const DataDisplay = () => {
    const { data, loading, error } = useFetch('https://api.example.com/data');

    if (loading) return <div>Loading...</div>;
    if (error) return <div>Error: {error.message}</div>;

    return (
        <div>
            {data && data.map(item => (
                <div key={item.id}>{item.name}</div>
            ))}
        </div>
    );
};

export default DataDisplay;
```

### Explanation
- **Data Fetching**: The custom hook `useFetch` abstracts the logic for fetching data, handling loading states, and error management, making it reusable across different components.
- **Cleaner Components**: With the custom hook, the `DataDisplay` component remains focused on rendering UI instead of handling side effects, leading to cleaner and more maintainable code.

Custom hooks help manage shared logic while keeping components clean and focused, enabling better organization and code reuse in your React applications.
