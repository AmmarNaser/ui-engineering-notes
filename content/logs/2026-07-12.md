### React Hooks - Managing State with useState

React hooks are functions that let you use state and other React features without writing a class. The `useState` hook is the most commonly used hook as it helps manage state in function components, promoting cleaner and more readable code.

### When to Use It
- **Managing State**: Use `useState` when you have a piece of state in your component that needs to be read and updated across renders.
- **Handling Form Inputs**: When creating forms, use `useState` to manage the values of form fields, making it easy to capture user input.

### Example
```jsx
import React, { useState } from 'react';

const Counter = () => {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
      <button onClick={() => setCount(count - 1)}>Decrement</button>
    </div>
  );
};

export default Counter;
```

### Explanation
- **Initial State**: The `useState` hook takes an initial value (in this case, `0`) and returns an array containing the current state and a function to update it.
- **State Updates**: You can update the state by calling the updater function (e.g., `setCount`). This triggers a re-render, allowing the component to reflect the new state.

Using `useState` enhances your ability to handle component state in a functional way, streamlining the process of building interactive UIs in React.
