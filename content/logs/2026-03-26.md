### useState - Managing State in Functional Components

The `useState` hook is a fundamental feature in React used to manage state within functional components. It allows developers to create and maintain local state without the need for class components, enabling a more streamlined and functional programming approach.

### When to Use It
- **Local State Management**: Use `useState` when you need to manage local state specific to a component, such as form inputs or toggles.
- **Dynamic Rendering**: Utilize `useState` to handle changes that trigger re-renders based on user interaction, like showing or hiding elements.

### Example
```jsx
import React, { useState } from 'react';

const Counter = () => {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
};

export default Counter;
```

### Explanation
- **State Initialization**: `useState` takes an initial state value, in this case `0`, and returns a state variable and a function to update it.
- **Reactivity**: Calling `setCount` triggers a re-render of the component, reflecting the updated state in the DOM.

Using `useState` effectively can greatly simplify component logic and enhance readability, making it a crucial tool for React developers.
