### Custom Hooks - The Power of Reusability in React

Custom hooks in React enable developers to encapsulate and reuse stateful logic across multiple components. By defining a hook, you can abstract away complex logic and share it easily, promoting cleaner code and better maintainability.

### When to Use It
- **Shared Logic**: When you find yourself repeating stateful logic across multiple components, consider creating a custom hook to consolidate that code into a single reusable function.
- **Complex Data Fetching**: If a component's data fetching logic is intricate and used in various places, you can simplify the process by creating a custom hook that handles the fetching and caching.

### Example
```javascript
import { useState, useEffect } from 'react';

// Custom Hook for data fetching
function useFetch(url) {
    const [data, setData] = useState(null);
    const [loading, setLoading] = useState(true);
    const [error, setError] = useState(null);

    useEffect(() => {
        const fetchData = async () => {
            try {
                const response = await fetch(url);
                if (!response.ok) {
                    throw new Error('Network response was not ok');
                }
                const result = await response.json();
                setData(result);
            } catch (error) {
                setError(error);
            } finally {
                setLoading(false);
            }
        };
        fetchData();
    }, [url]);

    return { data, loading, error };
}

// Example Component using custom hook
function DataDisplay({ url }) {
    const { data, loading, error } = useFetch(url);

    if (loading) return <p>Loading...</p>;
    if (error) return <p>Error: {error.message}</p>;

    return (
        <div>
            {data && <pre>{JSON.stringify(data, null, 2)}</pre>}
        </div>
    );
}
```

### Explanation
- **Reusability**: Custom hooks allow you to encapsulate and reuse logic, reducing duplication across your application and promoting DRY (Don't Repeat Yourself) principles.
- **Abstraction**: By abstracting complex logic into a hook, you make your components simpler and more focused on rendering, which improves readability and maintainability.

Custom hooks are a great way to promote cleaner code and reusability in React applications, making them an essential tool in any developer’s toolkit.
