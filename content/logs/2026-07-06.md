### State Management - Understanding React's useState Hook

State management is a fundamental concept in React that allows developers to manage and manipulate the application's state. The `useState` hook provides an easy way to create stateful components, enabling the development of dynamic and interactive applications.

### When to Use It
- **Single Value State**: When you need to maintain a simple piece of state, such as a boolean or a number, in a functional component.
- **Form Handling**: When building forms that require tracking of user input for fields like text boxes, checkboxes, or selects.

### Example
```jsx
import React, { useState } from 'react';

const Counter = () => {
  const [count, setCount] = useState(0); // Initialize state

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  );
};

export default Counter;
```

### Explanation
- **State Initialization**: The `useState` hook initializes the state variable. In this example, `count` starts at `0`.
- **State Update Function**: The function `setCount` updates the state when invoked. Each button click increases the count by `1`, triggering a re-render of the component.

By effectively utilizing the `useState` hook, developers can build intuitive UIs that respond to user actions seamlessly.
