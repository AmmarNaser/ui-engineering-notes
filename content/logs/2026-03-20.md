### Custom Hooks - Reusable Logic in React

Custom hooks in React allow you to extract component logic into reusable functions. This enables better organization of code and enhances reusability, making it more maintainable and reducing duplication across components.

### When to Use It
- **State Management**: When you need to manage state that is shared across multiple components, such as form inputs or fetching data.
- **Side Effects**: When working with side effects like subscriptions or timers that can be abstracted away from the components to improve readability and separation of concerns.

### Example
```javascript
import { useState, useEffect } from 'react';

// Custom Hook
function useFetch(url) {
    const [data, setData] = useState(null);
    const [loading, setLoading] = useState(true);
    const [error, setError] = useState(null);

    useEffect(() => {
        fetch(url)
            .then(response => {
                if (!response.ok) {
                    throw new Error('Network response was not ok');
                }
                return response.json();
            })
            .then(data => {
                setData(data);
                setLoading(false);
            })
            .catch(error => {
                setError(error);
                setLoading(false);
            });
    }, [url]);

    return { data, loading, error };
}

// Example Component
function DataFetchingComponent() {
    const { data, loading, error } = useFetch('https://api.example.com/data');

    if (loading) return <div>Loading...</div>;
    if (error) return <div>Error: {error.message}</div>;

    return (
        <div>
            {data && data.map(item => (
                <div key={item.id}>{item.name}</div>
            ))}
        </div>
    );
}
```

### Explanation
- **Reusability**: Custom hooks allow you to encapsulate logic that can be reused across different components, reducing code duplication.
- **Separation of Concerns**: By using custom hooks, you can keep your components clean and focused on rendering, while logic pertaining to state management and side effects is handled separately.

Custom hooks are a powerful feature in React that enhance code organization and promote reusability, making your applications easier to maintain.
