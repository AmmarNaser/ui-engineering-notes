### Custom Hooks - Creating Reusable Logic in React

Custom hooks allow you to extract and encapsulate reusable logic from your components in React. They help simplify code, improve organization, and promote the reuse of functionality across different components, making your React applications more maintainable.

### When to Use It
- **Stateful Logic Reusability**: When you have stateful logic that needs to be shared between components, custom hooks enable you to encapsulate this logic in a single place.
- **Complex Side Effects**: If a component manages complex side effects or interactions with external systems (like APIs or subscriptions), a custom hook can cleanly manage these without cluttering your component's code.

### Example
```jsx
import { useState, useEffect } from 'react';

function useFetch(url) {
    const [data, setData] = useState(null);
    const [error, setError] = useState(null);

    useEffect(() => {
        const fetchData = async () => {
            try {
                const response = await fetch(url);
                if (!response.ok) throw new Error('Network response was not ok');
                const result = await response.json();
                setData(result);
            } catch (error) {
                setError(error);
            }
        };

        fetchData();
    }, [url]);

    return { data, error };
}

// Usage Example in a Component
const App = () => {
    const { data, error } = useFetch('https://api.example.com/data');

    if (error) return <div>Error: {error.message}</div>;
    if (!data) return <div>Loading...</div>;

    return (
        <ul>
            {data.map(item => (
                <li key={item.id}>{item.name}</li>
            ))}
        </ul>
    );
};
```

### Explanation
- **Data Fetching Logic**: The `useFetch` custom hook encapsulates all the logic required to fetch data from an API, handling state for both the fetched data and potential errors.
- **Separation of Concerns**: By using custom hooks, we clearly separate the data fetching logic from the component rendering logic, making our components simpler and more focused.

Custom hooks are a powerful feature in React that can greatly enhance your application's code organization and reusability.
