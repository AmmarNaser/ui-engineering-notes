### Custom Hooks - Managing Component Logic Reusability

Custom hooks are a powerful feature in React that allow developers to extract and reuse stateful logic across multiple components. By encapsulating code into a hook, you can improve maintainability and reduce redundancy in your application.

### When to Use It
- **Component Logic Reuse**: When multiple components share the same logic—such as data fetching or form handling—it makes sense to create a custom hook to encapsulate that logic.
- **Complex State Management**: If a component has complex state logic that contains multiple state variables, using a custom hook will help separate the state management logic from the UI code, making your components cleaner and easier to understand.

### Example
```javascript
import { useState, useEffect } from 'react';

// Custom Hook
const useFetch = (url) => {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);
  
  useEffect(() => {
    const fetchData = async () => {
      try {
        const response = await fetch(url);
        if (!response.ok) throw new Error('Network response was not ok');
        const result = await response.json();
        setData(result);
      } catch (error) {
        setError(error);
      } finally {
        setLoading(false);
      }
    };

    fetchData();
  }, [url]);

  return { data, loading, error };
};

// Component using the custom hook
const DataDisplay = ({ url }) => {
  const { data, loading, error } = useFetch(url);

  if (loading) return <div>Loading...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div>
      <h3>Data:</h3>
      <pre>{JSON.stringify(data, null, 2)}</pre>
    </div>
  );
};
```

### Explanation
- **Reusability**: Custom hooks enable code reuse, allowing you to share common functionality across various components without rewriting the logic.
- **Separation of Concerns**: By utilizing custom hooks, you can separate the business logic from the presentation layer, which enhances the clarity and organizational structure of your code.

Using custom hooks can significantly improve your React applications by promoting clean code practices and making it easier to manage shared logic.
