### Hooks - Simplifying State Management in Functional Components

Hooks are a powerful feature in React that allow you to use state and other React features without writing a class. They enable functional components to manage and share state more easily, leading to cleaner and more reusable code.

### When to Use It
- **Managing State**: Use hooks when you need to manage local state in a functional component.
- **Side Effects**: Utilize hooks like `useEffect` for handling side effects such as data fetching or subscriptions.

### Example
```jsx
import React, { useState, useEffect } from 'react';

const Counter = () => {
  const [count, setCount] = useState(0);

  useEffect(() => {
    document.title = `Count is ${count}`;
  }, [count]);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  );
};

export default Counter;
```

### Explanation
- **useState**: This hook allows you to add state to functional components, returning a state variable and a function to update it.
- **useEffect**: This hook is used for side effects, allowing you to perform actions in response to state or prop changes, such as updating the document title in this example.

Using hooks can greatly enhance the structure and maintainability of your React applications while providing the benefits of stateful logic in functional components.
