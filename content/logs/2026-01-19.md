### Custom Hooks - Reusable State Logic in React

Custom hooks in React allow developers to extract component logic into reusable functions. This promotes clean code and makes it easier to maintain complex applications by encapsulating stateful logic and side effects that can be shared across components.

### When to Use It
- **State Logic Reuse**: When multiple components share similar stateful logic, custom hooks can help centralize and simplify the management of that logic.
- **Cleaner Component Structure**: When components become too large or complex, extracting logic into custom hooks can improve readability and maintainability.

### Example
```javascript
import { useState, useEffect } from 'react';

// Custom Hook
function useFetch(url) {
    const [data, setData] = useState(null);
    const [loading, setLoading] = useState(true);
    const [error, setError] = useState(null);
    
    useEffect(() => {
        const fetchData = async () => {
            try {
                const response = await fetch(url);
                if (!response.ok) {
                    throw new Error('Network response was not ok');
                }
                const result = await response.json();
                setData(result);
            } catch (error) {
                setError(error);
            } finally {
                setLoading(false);
            }
        };

        fetchData();
    }, [url]);

    return { data, loading, error };
}

// Component using the custom hook
function DataDisplay({ url }) {
    const { data, loading, error } = useFetch(url);

    if (loading) return <div>Loading...</div>;
    if (error) return <div>Error: {error.message}</div>;

    return <div>{JSON.stringify(data)}</div>;
}
```

### Explanation
- **Encapsulation of Logic**: Custom hooks encapsulate stateful logic and side effects, making it easy to reuse across multiple components without duplication.
- **Flexible Usage**: They can accept parameters and return whatever values are needed, providing flexibility for different scenarios while still maintaining the state management isolated.

By using custom hooks, developers can enhance code reusability and maintainable practices in their React applications.
