### State Management - Understanding React's useState Hook

State management is a critical aspect of React applications, allowing developers to create dynamic and interactive user interfaces. The `useState` hook provides a straightforward way to add state to functional components.

### When to Use It
- **Single Value State**: Use `useState` when you need to store and manipulate a single value, such as a user input or a toggle state.
- **Component Local State**: When you want to manage the local state of a component without affecting other components or the global state.

### Example
```jsx
import React, { useState } from 'react';

const Counter = () => {
  const [count, setCount] = useState(0);

  const increment = () => setCount(count + 1);
  const decrement = () => setCount(count - 1);

  return (
    <div>
      <p>Current Count: {count}</p>
      <button onClick={increment}>Increment</button>
      <button onClick={decrement}>Decrement</button>
    </div>
  );
};

export default Counter;
```

### Explanation
- **State Initialization**: The `useState` hook initializes the state variable (`count`) and provides a function (`setCount`) to update it.
- **Reactivity**: The component automatically re-renders whenever the state changes, reflecting the new count value on the UI.

The `useState` hook simplifies state management in functional components and is essential for creating responsive interfaces in React applications.
