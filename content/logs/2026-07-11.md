### Hooks - Managing State in Functional Components

Hooks are a powerful feature in React that allow you to use state and other React features without writing a class. They promote cleaner code and better organization, making it easier to manage component logic.

### When to Use It
- **State Management**: Use hooks for managing local state in functional components, allowing easier updates and reactivity.
- **Side Effects**: Apply hooks to perform side effects like data fetching or subscriptions in a clean and organized manner.

### Example
```jsx
import React, { useState, useEffect } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    document.title = `You clicked ${count} times`;
  }, [count]);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  );
}

export default Counter;
```

### Explanation
- **useState**: This hook allows functional components to manage state, providing a function to update the state variable.
- **useEffect**: This hook is used to perform side effects in function components, such as updating the document title when the `count` changes.

Incorporating hooks into your React components enhances code clarity and reusability, leading to a more maintainable codebase.
