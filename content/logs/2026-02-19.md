### Custom Hooks - Reusable Logic in React

Custom hooks are a powerful feature in React that allow developers to extract and reuse stateful logic across components. By encapsulating logic in a hook, we can enhance component reusability and maintainability while keeping our components clean and focused on rendering UI.

### When to Use It
- **Stateful Logic Sharing**: Use custom hooks when you have stateful logic that needs to be shared between multiple components, such as form handling or API fetching.
- **Code Organization**: Use when you want to organize complex component logic into separate functions, making your components easier to read and test.

### Example
```javascript
// useFetch.js
import { useState, useEffect } from 'react';

export const useFetch = (url) => {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const fetchData = async () => {
      try {
        const response = await fetch(url);
        if (!response.ok) throw new Error('Network response was not ok');
        const result = await response.json();
        setData(result);
      } catch (err) {
        setError(err);
      } finally {
        setLoading(false);
      }
    };
    fetchData();
  }, [url]);

  return { data, loading, error };
};

// Sample component using the custom hook
import React from 'react';
import { useFetch } from './useFetch';

const DataDisplay = ({ apiUrl }) => {
  const { data, loading, error } = useFetch(apiUrl);

  if (loading) return <div>Loading...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div>
      {data && data.map(item => (
        <div key={item.id}>{item.name}</div>
      ))}
    </div>
  );
};

export default DataDisplay;
```

### Explanation
- **Encapsulation**: Custom hooks encapsulate logic, allowing you to use it in any functional component without duplicating code.
- **Dependency Management**: You can manage dependencies for your effects inside the custom hook, ensuring that data is fetched correctly based on the parameters provided.

Custom hooks significantly simplify the management of complex logic in React applications and promote code reusability.
