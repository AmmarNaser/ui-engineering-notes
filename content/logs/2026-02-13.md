### Custom Hooks - Streamlining State Logic in React

Custom hooks in React allow you to encapsulate stateful logic in reusable functions. This concept enables developers to share logic across components without altering their component hierarchy, which promotes cleaner and more maintainable code.

### When to Use It
- **State Management**: When you have stateful logic that needs to be shared across multiple components, such as form handling or fetching data from an API.
- **Code Reusability**: When you want to abstract complex logic that is used in multiple places, minimizing code duplication and improving readability.

### Example
```jsx
import { useState, useEffect } from 'react';

function useFetch(url) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const fetchData = async () => {
      try {
        const response = await fetch(url);
        if (!response.ok) {
          throw new Error('Network response was not ok');
        }
        const result = await response.json();
        setData(result);
      } catch (err) {
        setError(err);
      } finally {
        setLoading(false);
      }
    };

    fetchData();
  }, [url]);

  return { data, loading, error };
}

export default useFetch;

// Usage in a component
import React from 'react';
import useFetch from './useFetch';

function DataFetchingComponent() {
  const { data, loading, error } = useFetch('https://api.example.com/data');

  if (loading) return <p>Loading...</p>;
  if (error) return <p>Error: {error.message}</p>;
  
  return (
    <div>
      <h1>Data:</h1>
      <pre>{JSON.stringify(data, null, 2)}</pre>
    </div>
  );
}
```

### Explanation
- **Reusable Logic**: Custom hooks allow you to extract and reuse component logic, keeping your components tidy.
- **Encapsulation of Effects**: Using custom hooks, you can encapsulate side effects related to data fetching for better separation of concerns.

Custom hooks enhance the power of React by promoting reusability and clean code practices, allowing developers to work more efficiently.
