### Custom Hooks - Reusable Logic in React

In React, custom hooks allow you to extract and reuse stateful logic across components, enhancing code maintainability and reducing duplication. They enable you to encapsulate complex behavior without compromising the simplicity of functional components.

### When to Use It
- **Shared Logic**: When you have logic that needs to be reused across multiple components, such as fetching data or managing a form state.
- **Complex State Management**: When managing intricate stateful logic that doesn’t naturally fit into the main component’s body, such as using side effects or timers.

### Example
```javascript
import { useState, useEffect } from 'react';

// Custom Hook
function useFetch(url) {
    const [data, setData] = useState(null);
    const [loading, setLoading] = useState(true);
    const [error, setError] = useState(null);

    useEffect(() => {
        const fetchData = async () => {
            try {
                const response = await fetch(url);
                if (!response.ok) throw new Error('Network response was not ok');
                const result = await response.json();
                setData(result);
            } catch (err) {
                setError(err);
            } finally {
                setLoading(false);
            }
        };

        fetchData();
    }, [url]);

    return { data, loading, error };
}

// Component using the custom hook
function DataDisplay({ apiUrl }) {
    const { data, loading, error } = useFetch(apiUrl);

    if (loading) return <div>Loading...</div>;
    if (error) return <div>Error occurred: {error.message}</div>;

    return (
        <div>
            <h1>Data:</h1>
            <pre>{JSON.stringify(data, null, 2)}</pre>
        </div>
    );
}
```

### Explanation
- **Code Reusability**: Custom hooks enable the reuse of complex logic without repeating code, adhering to the DRY (Don't Repeat Yourself) principle.
- **Separation of Concerns**: They help maintain a clear separation between component logic and presentation, making the code easier to understand and test.

By using custom hooks, you can enhance your React applications by making them more modular and maintainable.
