### Custom Hooks - Reusable Logic in React

Custom Hooks in React allow developers to encapsulate and reuse stateful logic across multiple components, promoting code reusability and maintainability. They help keep the component code cleaner and more focused on rendering UI.

### When to Use It
- **Shared Logic**: When you have a stateful logic that needs to be shared across multiple components, such as form handling or data fetching.
- **Complex State Management**: When managing complex state that would benefit from being abstracted away from the component, improving readability and separation of concerns.

### Example
```javascript
import { useState, useEffect } from 'react';

// Custom hook for fetching data
function useFetch(url) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const fetchData = async () => {
      try {
        const response = await fetch(url);
        if (!response.ok) throw new Error('Network response was not ok');
        const result = await response.json();
        setData(result);
      } catch (err) {
        setError(err);
      } finally {
        setLoading(false);
      }
    };
    fetchData();
  }, [url]);

  return { data, loading, error };
}

// Example usage of the custom hook in a component
function DataComponent() {
  const { data, loading, error } = useFetch('https://api.example.com/data');

  if (loading) return <p>Loading...</p>;
  if (error) return <p>Error: {error.message}</p>;
  
  return <div>{JSON.stringify(data)}</div>;
}
```

### Explanation
- **Reusability**: Custom Hooks can be reused across different components, reducing duplication and making code easier to maintain.
- **Abstraction**: They can abstract complex logic outside the component, keeping the component itself more focused on rendering and user interactions.

Custom Hooks are an essential tool in the React developer’s toolkit, enabling cleaner and more maintainable code by encapsulating shared logic.
